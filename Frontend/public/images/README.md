# Imágenes de Fondo

Coloca aquí tus imágenes de fondo para la aplicación.

## Instrucciones:

1. **Agrega 5 imágenes** con los siguientes nombres:
   - `bg1.jpg`
   - `bg2.jpg`
   - `bg3.jpg`
   - `bg4.jpg`
   - `bg5.jpg`

2. **Recomendaciones:**
   - Resolución mínima: 1920x1080px
   - Formato: JPG o PNG
   - Imágenes relacionadas con: profesionales, tecnología, negocios, oficinas, etc.
   - Preferiblemente con colores azules/índigo que combinen con el tema

3. **Funcionalidad:**
   - Las imágenes rotarán automáticamente cada 8 segundos
   - Cada imagen hace un zoom out suave al aparecer
   - Hay un overlay de color para mejorar la legibilidad

## Opcional: Usar más o menos imágenes

Si quieres usar más o menos de 5 imágenes:

1. Abre `Frontend/src/App.jsx`
2. Busca el array `backgroundImages`
3. Agrega o quita rutas según necesites:

```javascript
const backgroundImages = [
  '/images/bg1.jpg',
  '/images/bg2.jpg',
  '/images/bg3.jpg',
  // Agrega más aquí...
];
```
